VIBE.TUNER is a web app developed to connect different users around the world through music and feelings. This app has two main features. On one side the user can advise a song which is associated with a mood chosen by the user himself. Once sent, the song is ready to be recommended to another user with the same mood. We developed it using Spotify API, Google Firebase and P5.js.
